# Auto-Chapter Video Generator

A lightweight, local command-line tool that automatically generates YouTube-compatible video chapters from any YouTube video URL. 

The application downloads a short segment of a video, processes it using the specialized Hugging Face Vision-Language Model (VLM) `NemoStation/Marlin-2B` (a 2B parameter Qwen2.5-VL fine-tune specialized in video temporal grounding), and writes the chapter events directly to `chapters.txt`.

---

## Tech Stack

- **Core VLM**: [NemoStation/Marlin-2B](https://huggingface.co/NemoStation/Marlin-2B) (built on Qwen2.5-VL-2B-Instruct)
- **Video Download/Cut**: `yt-dlp` (integrated with system `FFmpeg`)
- **VLM Pipeline**: Hugging Face `transformers`, `qwen-vl-utils`, `accelerate`, and `torch` (CUDA optimized)

---

## Prerequisites

Before running the application, make sure you have:
1. **Python 3.10+** installed.
2. **Git** installed.
3. **FFmpeg** installed and added to your system's PATH. 
   - FFmpeg is *required* by `yt-dlp` to cut the video dynamically during download.
   - *Tip (PowerShell installation via Winget):* `winget install Gyan.FFmpeg`
4. **Hugging Face Account & Read Token**: Required to download the gated Marlin-2B model weights.

## Setup Steps

Open Windows PowerShell and run the following commands:

```powershell
# 1. Clone or navigate to the workspace directory
cd "d:\Ray Codes\AG Projects\Merlin 2B"

# 2. Create a Python virtual environment (venv)
python -m venv venv

# 3. Activate the virtual environment
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
.\venv\Scripts\Activate.ps1

# 4. Upgrade pip and install the dependencies
python -m pip install --upgrade pip
pip install -r requirements.txt

# 5. Create a .env file and set your Hugging Face Token:
# HF_TOKEN=your_token_here
```

---

## Run Steps

With your virtual environment activated, run the script:

```powershell
# Run the application with the default or custom YouTube URL
python app.py "https://www.youtube.com/watch?v=yoa4opGsvP4"
```

## Test Steps (How to Verify It Worked)

1. **Verify Download**: Check that `test_video.mp4` was downloaded in the root folder. You can verify its duration is up to 2 minutes.
2. **Check Hugging Face Cache**: The model `NemoStation/Marlin-2B` will download to your local Hugging Face cache folder during the first run.
3. **Inspect Chapters Output**: Check that `chapters.txt` was created in the workspace. It should contain timestamps and event descriptions generated by Marlin-2B.

## Real-World Use Cases

- **Automated YouTube Uploads**: Instantly generate timestamps and titles for uploaded lectures, tutorials, or gameplay videos to improve viewer navigation.
- **Short-Form Content Creation**: Scan raw footage or stream recordings to locate interesting highlights with exact start-end timestamps.
- **Course & Lecture Indexing**: Help students navigate online courses by breaking down hour-long lectures into distinct, searchable topics.
- **Podcast Summarization**: Segment audio/video podcasts into logical topic blocks automatically for social media promotions.
- **Vlog Chapters**: Automate the process of adding chapters to vlogs without manually scrubbing through timelines.
